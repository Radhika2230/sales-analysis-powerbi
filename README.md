# sales-analysis-powerbi

**Sales & Profit Analysis Dashboard (Power BI)**

**_Overview_**
This project is an end-to-end Power BI dashboard built using retail sales data to analyze sales performance, profitability, and trends.

🚀 Key Features
- KPI tracking (Sales, Profit, Profit Margin)
- Month-over-Month Growth Analysis
- Top Product Analysis
- Regional Performance
- Category Insights
- Interactive slicers for filtering

🧠 Data Modeling
- Implemented Star Schema
- Created a dedicated Date Table
- Established relationships for time intelligence

📊 DAX Measures
- Total Sales
- Total Profit
- Profit Margin %
- Sales Growth %

🐍 Python Integration
- Used Python visual to implement Moving Average trend
- Enhanced analysis beyond standard visuals

🛠 Tools Used
- Power BI
- Power Query
- DAX
- Python (pandas, matplotlib)

🎯 Business Impact
This dashboard helps identify:
- High-performing products
- Profitable regions
- Sales trends and growth patterns
